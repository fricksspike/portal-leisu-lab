# portal-leisu-lab

## 项目简介

这是一个用于归档和发布多个独立 HTML 页面的仓库。本仓库不针对任何特定域名或网站，仅作为页面内容的集中存放与版本管理使用。

## 目录说明

```
portal-leisu-lab/
├── pages/          # 存放各独立 HTML 页面文件
├── assets/         # 页面引用的静态资源（如样式、脚本、图片等）
├── README.md       # 本文件
└── LICENSE         # 仓库许可证（如有）
```

- `pages/` 目录下每个 HTML 文件代表一个独立的归档页面，文件名通常反映页面内容或用途。
- `assets/` 目录用于存放页面所需的辅助资源，按需组织子目录。

## 页面归档说明

- 所有页面均为纯静态 HTML，不依赖后端服务。
- 页面内容可能涉及多种主题，彼此之间无直接关联。
- 每个页面的创建与更新均通过 Git 提交记录追踪，便于回溯历史版本。

## 维护说明

- 欢迎提交 Pull Request 添加新的归档页面或改进现有页面。
- 提交前请确保页面自包含，无外部依赖（或已正确引用 `assets/` 中的资源）。
- 若需删除或修改页面，建议在 Issue 中先进行讨论。
- 本仓库遵循通用开源协议，具体条款见 `LICENSE` 文件。

## 使用方式

1. 克隆仓库到本地：
   ```bash
   git clone https://github.com/your-username/portal-leisu-lab.git
   ```
2. 直接在浏览器中打开 `pages/` 目录下的 HTML 文件预览。
3. 也可通过 GitHub Pages 等服务部署，但本仓库不预设任何部署配置。
